# melaniemcd-ai

This repository contains the source code for my personal portfolio website. I publish it primarily for transparency and to share implementation patterns.

## Live site

* [https://melaniemcd.ai](https://melaniemcd.ai)
* Hosting: Replit

## What you will find here

* Portfolio content and case notes
* Demos and prototypes focused on practical AI adoption in local government workflows
* Reusable UI patterns and small experiments
* Writing and reflections (personal)

## Tech stack

* Framework: Next.js
* Language: TypeScript
* Styling: Tailwind CSS
* Hosting: Replit

## Notes on use

This is a personal portfolio codebase. You are welcome to browse the code. It is not intended to be used as a template and it is not maintained as a general-purpose package.
